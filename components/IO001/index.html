    TTP223 - IO001¶

    A tiny one‑key capacitive touch switch module based on the TTP223 IC. Acts like a digital push‑button: when you touch the pad it toggles or holds the output depending on solder‑pad settings. Ideal as a human‑touch input for ESP32 projects (UI buttons, lamps, mode selectors). Power it at 3.3 V when connecting directly to ESP32 GPIO.

    Links¶

    • Where to buy: AliExpress
    • Datasheet: PDF
    • Tutorial: ESP32 + TTP223 (Instructables)

    Specifications¶

    • Module size: 15 mm × 11 mm
    • Supply voltage (VCC): 2.5–5.5 V
    • Output: digital TTL (follows VCC), active HIGH or LOW per configuration
    • Modes (A/B solder pads):
    • A=0, B=0 → momentary, output HIGH while touched
    • A=1, B=0 → toggle (self‑latch), output HIGH toggles on each touch
    • A=0, B=1 → momentary, output LOW while touched
    • A=1, B=1 → toggle (self‑latch), output LOW toggles on each touch

    Pinout & Addresses (common breakout labels)¶

    • VCC — 2.5–5.5 V supply (use 3.3 V for ESP32 direct GPIO)
    • OUT — Digital output (HIGH/LOW per mode)
    • GND — Ground
    • Pads A/B — Solder‑jumpers selecting momentary vs toggle and active‑high vs active‑low (see table above).

    Wiring¶

    ESP32 (3.3 V): - VCC → 3V3 - GND → GND - OUT → any GPIO (e.g., GPIO 25)

    No extra resistors are required on typical boards; the module conditions the signal. If you enable the ESP32's internal pull‑down or pull‑up, adjust logic accordingly.

    Gotchas¶

    • If you power the module at 5 V, OUT will be 5 V and not 3.3 V‑safe → either power at 3.3 V or level‑shift before an ESP32 GPIO.
    • Long leads, moisture, or noisy grounds can cause false triggers—keep wiring short and add decoupling (e.g., 0.1 µF across VCC–GND) if needed.
    • Touch pad sensitivity varies by board and environment; avoid mounting behind thick plastic or metal without testing.

    How to use¶

    Minimal example (Arduino IDE on ESP32). Assumes A=0,B=0 (momentary, active‑HIGH) and OUT on GPIO 25:

    // TTP223 -> ESP32 example
// VCC=3V3, GND=GND, OUT=GPIO25
const int TOUCH_PIN = 25;

void setup() {
  pinMode(TOUCH_PIN, INPUT);      // or INPUT_PULLDOWN if you prefer
  Serial.begin(115200);
}

void loop() {
  int touched = digitalRead(TOUCH_PIN); // HIGH while finger is touching
  if (touched) {
    Serial.println("Touch!");
  }
  delay(10);
}
