# Tact switch kit (25 types, 125pcs) - SW002

Assorted momentary tactile push buttons in common SMD and DIP sizes (e.g., 3×6, 6×6, 12×12). Useful for quick ESP32 input prototypes and replacements.

## Links

- **Where to buy:** [AliExpress](https://www.aliexpress.com/item/1005004341295362.html)


## Specifications

- Quantity: 25 types × 5 pcs = 125 pcs total

- Action: Momentary, normally open (NO)

- Packages: Mixed SMD and through‑hole (DIP) footprints


## Pinout & Addresses (common breakout labels)

- Typical 4‑pin tact switch: opposite pins are internally common; use one pin from each common side.

- 2‑pin micro tact: the two leads are the switch contacts.


## Wiring

- Wire one side to ESP32 GPIO (enable INPUT_PULLUP) and the other to GND.

- Debounce in software (e.g., 10–50 ms).
