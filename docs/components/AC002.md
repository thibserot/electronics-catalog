# Ultra-thin Horn Speaker - AC002

A very small **30 mm, 8 Ω, 0.5 W** ultra‑thin dynamic speaker. Use it for simple beeps, prompts, or low‑volume audio in compact projects. **It cannot be driven directly from an ESP32 pin** — pair it with a small amplifier (e.g., LM386 or a class‑D module like PAM8302).

## Specifications

- Impedance: **8 Ω (8R)**
- Power rating: **0.5 W**
- Diameter: **30 mm** (ultra‑thin style)
- Leads: **2‑wire pigtail** (vendor notes “P2.0 termina wire”)  
*(Values taken from the AliExpress listing; exact SPL/response not provided.)*

## Pinout & Addresses (common breakout labels)

Not applicable — two wires only. **Polarity is not critical for a single speaker**, but keep consistent (+/–) if using multiple speakers to avoid phase issues.

## Wiring

**Do not** connect the speaker to an ESP32 GPIO directly. Use a small amplifier:

- **LM386** (simple, through‑hole; 5 V supply; class‑AB)
- **PAM8302** (efficient class‑D; 2.5–5.5 V; recommended for battery projects)

Example with **LM386** (ESP32 DAC → LM386 → speaker):

- ESP32 **GPIO25 (DAC1)** → 100 nF (series) → LM386 **IN**
- LM386 **VCC** → 5 V; **GND** → GND
- Gain: leave pins **1–8** open for ×20, or add 10 µF between 1–8 for higher gain (per datasheet)
- Speaker to LM386 **OUT** and **GND**
- Add 220 µF from **OUT** to speaker if using a bare IC per typical app (see datasheet)

## Gotchas

- **Never** drive the speaker from a GPIO directly — you will damage the ESP32 or get inaudible output.
- LM386 is easy but **inefficient** and can hiss at high gain; use **PAM8302** for low‑power builds.
- Keep leads short and twist them to reduce noise pickup.
- This 30 mm, 0.5 W unit is for **low‑volume** use; don’t over‑drive.

## How to use

Minimal ESP32 sketch that generates a tone via PWM (LEDC) into an **amplifier input**:

```cpp
// ESP32 PWM beep -> feed amplifier input (LM386/PAM8302) via 0.1 uF cap
// No extra libraries required.
const int AUDIO_PIN = 25; // or another GPIO

void setup() {
  ledcSetup(0, 2000, 8);   // channel 0, 2 kHz, 8-bit
  ledcAttachPin(AUDIO_PIN, 0);
}

void beep(unsigned freq, unsigned ms) {
  ledcWriteTone(0, freq);
  delay(ms);
  ledcWriteTone(0, 0); // stop
}

void loop() {
  beep(1000, 200);  // 1 kHz beep
  delay(800);
}
```

For WAV/streamed audio using the ESP32 **DAC** and an LM386, see the linked tutorial.
